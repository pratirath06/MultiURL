## Usage

0. Clone the repository:
    ```bash
   https://github.com/pratirath06/MultiURL.git
    ```

1. Navigate to the directory
    ```bash
    cd MultiURL
    ```

2. Download and install the dependencies
   ```bash
   pip install -r requirements.txt
   ```

3. Run the python script
   ```bash
   python main.py
   ```
